المقدمة والتعريف بـ CSS

CSS (Cascading Style Sheets) هي لغة تنسيق تُستخدم لتنسيق وتصميم صفحات الويب. تمكنك من التحكم في مظهر عناصر HTML من حيث اللون والخط والحجم والموضع وغيرها.

الوظيفة الأساسية لـ CSS هي تنسيق عناصر صفحات الويب التي تم بناؤها بـ HTML، فإذا كان HTML هو هيكل الموقع، فإن CSS هو مظهره وتصميمه.

قبل البدء في كتابة أكواد CSS، يجب أن تقوم أولاً بإنشاء موقع الويب الخاص بك:

  1. إنشاء مجلد (folder) لمشروع الموقع يحتوي على جميع الملفات المطلوبة.
  2. فتح المجلد في محرر الأكواد (مثل Visual Studio Code).
  3. إنشاء ملف HTML كخطوة أولى لبناء هيكل الموقع.

الطريقة الأولى: كتابة CSS داخل ملف HTML (Internal CSS)

الطريقة الأولى لكتابة CSS هي إضافة أكواد CSS مباشرة داخل ملف HTML باستخدام وسم <style> داخل قسم <head>.

كيفية استخدام الـ Internal CSS:

<head>
  <style>
    h2 {
      color: red;
      font-size: 50px;
    }

    p {
      color: blue;
    }
  </style>
</head>
ملاحظة هامة: عند استخدام هذه الطريقة، فإن التنسيقات المكتوبة ستطبق فقط على الصفحة الحالية.

مميزات وعيوب الـ Internal CSS:

المميزات

  • سهولة التنفيذ والتعديل
  • عدم الحاجة لملفات إضافية
  • مناسب للصفحات الصغيرة

العيوب

  • زيادة حجم ملف HTML
  • عدم إمكانية إعادة استخدام الأكواد في صفحات أخرى
  • صعوبة الصيانة في المواقع الكبيرة

مثال عملي

عند كتابة الكود التالي:

<style>
  h2 {
    color: red;
  }
</style>

<h2>عنوان بلون أحمر</h2>

سيظهر العنوان h2 باللون الأحمر في الصفحة.

الطريقة الثانية: كتابة CSS في ملف خارجي (External CSS)

الطريقة الثانية والأكثر استخداماً في المشاريع الحقيقية هي كتابة أكواد CSS في ملف خارجي منفصل بامتداد .css ثم ربطه بملف HTML باستخدام وسم <link>.

خطوات استخدام الـ External CSS:

  1. إنشاء ملف CSS
    قم بإنشاء ملف جديد بامتداد .css (مثل style.css)
  2. كتابة أكواد CSS
    اكتب أكواد CSS في الملف بنفس الطريقة التي تكتب بها في وسم <style>
  3. ربط ملف CSS بصفحة HTML
    استخدم وسم <link> داخل قسم <head> لربط ملف CSS بصفحة HTML

كود ربط ملف CSS بصفحة HTML:

<head>
  <link rel="stylesheet" href="style.css">
</head>
شرح الكود:
rel="stylesheet": تحدد نوع الملف (ملف تنسيق)
href="style.css": مسار ملف CSS (اسم الملف إذا كان بنفس المجلد)

ربط ملف CSS في مجلد آخر

إذا كان ملف CSS في مجلد آخر (مثل مجلد css)، فإن كود الربط سيكون:

<link rel="stylesheet" href="css/style.css">

مميزات استخدام External CSS

  • فصل هيكل HTML عن التنسيق
  • إمكانية استخدام نفس ملف CSS لعدة صفحات HTML
  • سهولة الصيانة والتعديل
  • تقليل حجم صفحات HTML
  • تحسين أداء الموقع عبر تخزين ملف CSS للزيارات المتكررة

الطريقة الثالثة: كتابة CSS مباشرة داخل الوسم (Inline CSS)

الطريقة الثالثة هي كتابة أكواد CSS مباشرة داخل وسم HTML باستخدام سمة style. تُطبق هذه التنسيقات فقط على العنصر الذي تمت إضافتها له.

كيفية استخدام الـ Inline CSS:

<h2 style="color: red; font-size: 40px;">عنوان باللون الأحمر</h2>
<p style="color: blue;">فقرة باللون الأزرق</p>
ملاحظة هامة: عند استخدام هذه الطريقة، يُكتب الكود مباشرة داخل الوسم دون الحاجة لتحديد العنصر (لأنك بالفعل داخل العنصر).

كيفية إضافة عدة خصائص:

يمكنك إضافة أكثر من خاصية في نفس السمة style بفصلها باستخدام الفاصلة المنقوطة (;):

<h2 style="color: red; font-size: 40px; font-weight: bold;">
  عنوان بعدة تنسيقات
</h2>

متى تستخدم Inline CSS

  • تغيير عنصر واحد فقط
  • تجاوز التنسيقات الأخرى
  • إجراء اختبارات سريعة

عيوب Inline CSS

  • صعوبة الصيانة
  • تكرار الكود
  • زيادة حجم HTML
تنبيه: لا ينصح باستخدام هذه الطريقة بشكل أساسي في بناء المواقع، فهي تجعل الكود أكثر تعقيداً وأصعب في الصيانة.

مثال عملي

الكود:

<h2 style="color: red;">عنوان باللون الأحمر</h2>
<p style="color: blue; font-size: 20px;">فقرة باللون الأزرق</p>

تركيبة كود CSS الأساسية

لفهم طريقة كتابة CSS بشكل صحيح، يجب أن تتعرف على تركيبة الكود الأساسية.

selector {
  property1: value1;
  property2: value2;
  /* يمكنك إضافة المزيد من الخصائص */
}

شرح مكونات كود CSS:

المكون المصطلح الإنجليزي الوصف مثال
المُحدد Selector يحدد العنصر الذي تريد تنسيقه h2, p, .class, #id
الخاصية Property الجانب الذي تريد تغييره في العنصر color, font-size, margin
القيمة Value القيمة المحددة للخاصية red, 20px, bold
النقطتان Colon (:) تفصل بين الخاصية وقيمتها color: red;
الفاصلة المنقوطة Semicolon (;) تفصل بين الخصائص المختلفة color: red; font-size: 20px;
معلومة هامة: يمكنك إضافة عدد غير محدود من الخصائص للعنصر الواحد، ولكن يجب الفصل بينها بالفاصلة المنقوطة (;).

مثال شامل:

h2 {
  color: red;
  font-size: 50px;
  font-weight: bold;
  text-align: center;
}

في المثال السابق، اخترنا جميع عناصر h2 وطبقنا عليها أربع خصائص مختلفة.

يمكن تشبيه تركيبة CSS كما لو كنت تصف شخصاً ما:

  • المُحدد (Selector): الشخص الذي تصفه (مثل محمد)
  • الخاصية (Property): الصفة التي تصفها (مثل الطول)
  • القيمة (Value): قيمة هذه الصفة (مثل 180 سم)

  • دائماً أضف الفاصلة المنقوطة (;) بعد كل خاصية
  • استخدم المسافات لجعل الكود أكثر قابلية للقراءة
  • يمكنك كتابة تعليقات في CSS باستخدام /* تعليق */
  • تجنب تكرار الأكواد باستخدام الـ External CSS

مثال بصري

SelectorPropertyValue

الخلاصة

تعلمنا في هذا الدرس الطرق الثلاث الأساسية لكتابة أكواد CSS وهي:

  1. Internal CSS: كتابة أكواد CSS داخل ملف HTML باستخدام وسم <style>
  2. External CSS: كتابة أكواد CSS في ملف خارجي منفصل وربطه بملف HTML
  3. Inline CSS: كتابة أكواد CSS مباشرة داخل وسم HTML باستخدام سمة style

الطريقة المفضلة في معظم المشاريع هي External CSS نظراً لفوائدها المتعددة في تنظيم الكود وإعادة استخدامه.

العودة للبداية

أسئلة شائعة

أفضل طريقة هي استخدام External CSS (ملف خارجي) لأنها:

  • تفصل بين هيكل HTML والتنسيق
  • تسمح بإعادة استخدام نفس الأكواد في عدة صفحات
  • تسهل صيانة وتعديل الموقع
  • تحسن أداء الموقع عبر تخزين ملف CSS

يُفضل استخدام Inline CSS في الحالات التالية:

  • عند الحاجة لتغيير تنسيق عنصر واحد فقط
  • لتجاوز التنسيقات الأخرى (لأن Inline CSS له أعلى أولوية)
  • عند إجراء اختبارات سريعة
  • في بعض حالات البريد الإلكتروني HTML حيث لا يمكن استخدام ملفات خارجية

أولوية تطبيق CSS من الأعلى للأدنى:

  1. Inline CSS (مباشرة في العنصر)
  2. Internal CSS (داخل ملف HTML)
  3. External CSS (في ملف خارجي)

هذا يعني أن التنسيقات المكتوبة بطريقة Inline ستطغى على التنسيقات المكتوبة في Internal أو External CSS.

بعد تعلم أساسيات CSS، يمكنك الانتقال إلى المواضيع التالية:

  • خصائص CSS المختلفة (الألوان، الخطوط، الهوامش، الحدود، إلخ)
  • CSS Box Model (النموذج الصندوقي)
  • CSS Positioning (التموضع)
  • CSS Flexbox و CSS Grid
  • CSS Media Queries (للمواقع المتجاوبة)
  • CSS Animations (التحريكات)
